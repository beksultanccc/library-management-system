
Университеттік кітапхана ақпараттық жүйесі (ULMS)

 Сипаттама

Бұл жоба университет кітапханасын басқаруға арналған заманауи веб-қолданба. 
Жүйе студенттерге, оқытушыларға және әкімшілік персоналға кітап қорын басқару, 
онлайн тапсырыс/қолдану, оқу залы броньдау, қарыздар мен айыппұлдарды бақылау мүмкіндігін береді.

 Мүмкіндіктер

- ✅ Пайдаланушыларды тіркеу/аутентификация (JWT токендерімен)
- ✅ Кітаптарды іздеу және сүзу (Elasticsearch арқылы)
- ✅ Кітап картасы - қолжетімділік, көшірме саны, орналасқан орын
- ✅ Кітапты қарызға алу/қайтару операциялары
- ✅ Қарыздарды және мерзімдерді басқару
- ✅ Кітаптарды резервтеу
- ✅ Хабарламалар (email/SMS) - мерзім өтемесі, резерв орындалды

## Технологиялар

- **Backend**: FastAPI (Python 3.11)
- **Дерекқор**: PostgreSQL 15
- **Кэш**: Redis 7
- **Іздеу қозғалтқышы**: Elasticsearch 8.x
- **Хабарлама брокері**: RabbitMQ
- **Контейнерлеу**: Docker, Docker Compose
- **Миграция**: Alembic
- **Тестілеу**: pytest

 бастау

1. Қосу және бастау Docker Compose:
   ```
   docker-compose up --build
   ```
   The API will be available at http://localhost:8000
```

2. Interactive docs:
   - OpenAPI: http://localhost:8000/docs
   - Redoc: http://localhost:8000/redoc
